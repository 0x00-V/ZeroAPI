Users

GET /users — List all users

POST /users/register — Register a user curl -X POST /users/register \ -H "Content-Type: application/json" \ -d "{\"Name\":\"test\",\"Password\":\"test\"}"

POST /users/login — Authenticate as a user curl -X POST /users/login \ -H "Content-Type: application/json" \ -d "{\"Name\":\"test\",\"Password\":\"test\"}"
